¿Cómo publicar masivamente?

Paso 1: Selecciona que tipo de publicación realizarás (productos o servicios)

Paso 2: Descarga la planilla de excel para poder completar los datos de tus publicaciones (Tienes que tener el programa Microsoft Excel en tu computadora). Descargar Microsoft Excel

Paso 3: Completa los datos de tus publicaciones en la planilla. Una vez terminado tienes que subir el archivo excel. Nosotros lo revisaremos y te notificaremos una vez que esté aprobado.
