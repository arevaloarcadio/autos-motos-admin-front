¿Cómo publicar masivamente?

Paso 1: Descarga la planilla CSV para poder completar los datos de tus publicaciones (Tienes que tener el programa Microsoft Excel en tu computadora). Descargar Microsoft Excel

Paso 2: Completa los datos de tus publicaciones en la planilla. Una vez terminado tienes que subir el archivo CSV. Nosotros lo revisaremos y te notificaremos una vez que esté aprobado.

Puedes descargar un archivo de ejemplo para ver como se completan los datos.
